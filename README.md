# 🍳 Ovenwhisk  – React Frontend

Ovenwhisk LMS is a modern and responsive **frontend web application** built using **React.js**.  
It provides an intuitive user interface for students and instructors to access courses, manage learning content, and interact with the Learning Management System.

This project was bootstrapped using **Create React App**.

---

## 🚀 Features

- 🔐 User Authentication UI (Login / Register)
- 📚 Course Listing and Details
- 👨‍🏫 Instructor Dashboard
- 🎓 Student Dashboard
- 📂 Learning Material Viewer
- 📝 Assignment Interface
- 📊 Progress and Performance View
- 📱 Responsive Design

---

## 🛠️ Tech Stack

- React.js
- JavaScript (ES6+)
- HTML5
- CSS3
